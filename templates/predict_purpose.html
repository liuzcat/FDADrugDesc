Type in comma-separated active and/or inactive ingredients of your drug to see what purpose it gets sorted into.


About

The current model loaded for the ingredients panel is a simple Dense-layered (tensorflow/keras) neural network with ~91% accuracy. This was the highest accuracy thus far with some parameter tuning among different models.

The comprehensive average accuracies tested using 5-fold cross validation will be posted soon.

Techniques
Neural Network Linear Support Vector Classification Random Forest Multinomial Naive Bayes

There were 64 categories of purpose drawn from the dataset that each had over 200 drugs attributed to them. This is certainly not a complete picture of all FDA-approved drugs or even FDA-approved drugs with labeled purposes.
